Directives Example:
Structural Directive: Add/Remove elements from the DOM

*ngIf -> Server was created, server name is {{ serverName}}

*ngIf/else -> Server was created, server name is {{ serverName }}

ngIf/else -> No server was created!

Attribute Directives: Change the element they're placed on. Property binding and directives aren't the same thing (even though they both use [] for notation)


[ngStyle]=backgroundColor: getColor()


[ngClass]=serverCreated
